P0 合作方管理

  • 合作方列表 — GET /internal/admin/partners
  • 新建 / 编辑 — POST / PUT
  • 详情 Tab：基本信息、能力、凭证、回调

  • 调用列表 — GET /internal/admin/invocations
  • 调用详情 Drawer — requestId / 错误 / taskId

  • 投递日志列表 — 按 Partner / 事件类型
  • 重试状态、HTTP 状态码
  • Partner 详情 · Webhook Tab 扩展

P2 运营看板

  • 全局调用量 / 成功率 / Top 错误码
  • 按 Partner / API 维度图表
  • GET .../partners/{id}/stats

P2 API 目录

  • api_operation 只读列表
  • 能力码 / 路径 / 方法对照 OpenAPI

P2 配额与限流

  • Partner 级 QPS / 日配额报表
  • 429 触发统计

P3 API 版本管理

  • 版本发布 / 弃用时间表
  • 影响 Partner 通知

P3 告警规则

  • 错误率 / 延迟阈值
  • 通知渠道配置
